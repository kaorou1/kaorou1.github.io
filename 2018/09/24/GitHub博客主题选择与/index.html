GitHub博客更新与配置主题

更新博客

  1. hexo new “blog title”，然后编辑对应的md文件
  2. hexo s
  3. hexo g
  4. hexo deploy

主题配置流程

  1. 官方的主题商店，官方hexo主题大全，里面有主题展示的博客主页，选择一个自己喜欢的。

  2. 找到对应的Github链接，例如papaerbox主题，下载到本地。

  3. 将下载得到的文件夹放入到blog中的themes文件夹中。
  4. 修改主题文件夹名称（主要是改短一点），例如paperbox。 然后打开站点配置文件，找到 theme 字段，并将其值更改为 主题文件夹名称。
  5. 针对主题中的一些个性化配置，需要参考对应主题的readme，修改主题文件夹中的_config.yml文件。
  6. hexo clean, hexo g, hexo s, hexo deploy, 稍等片刻即可以看到网站效果。
